A collection of scripts or Jupyter notebooks that I have written to compute invariants, or values of formulas for known quantites. I have written them for myself but on the off chance that someone else wants to use them I will try to make them readable, explain what they do and how to use them. 

## Included
- A sage math notebook that takes in an analytic function and a degree and returns the element of the multiplicative sequence associated to that power series in that degree. The process described is that of Milnor-Stasheffs book "Characteristic classes". The multiplicative sequence is thought of as a polynomial in the Pontryagin classes and hence as a genus. This same code contains the slight change of perspective to additive sequences that allow the computation of so called characters, importatly the Chern and Pontryagin characters.
